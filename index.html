Tutorial

How to Take the Quiz


This is a randomised multiple-choice quiz. For each question, select the answer you think is correct and then click the "Next" button to proceed to the next question. Green answers are correct, while Red answers are incorrect.


You can only select one answer per question, so choose carefully! After you select an answer, a fun fact about cats will be displayed below. There are only 8 questions in total, and each question is only worth 1 point. At the end of the quiz, you'll see your score. Good luck!
